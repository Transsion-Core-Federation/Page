# Penyalahgunaan Kekuasaan Admin

**Aturan**: Pengelola (Admin) dilarang menggunakan wewenangnya untuk kepentingan pribadi atau melakukan tindakan disiplin (seperti Fban) terhadap anggota tanpa alasan yang objektif.

**Konsekuensi**: Kick, Ban, atau Fban.

**Penjelasan**: Kami menjunjung tinggi keadilan. Wewenang admin diberikan untuk menjaga ketertiban grup, bukan untuk penyalahgunaan kekuasaan demi kepentingan individu atau kelompok tertentu.
