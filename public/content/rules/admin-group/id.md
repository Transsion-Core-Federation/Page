# Admin Grup

**Aturan**:
- **Melapor ke Admin Federasi**: Admin grup wajib melaporkan setiap pelanggaran aturan oleh anggota kepada pihak Admin Federasi melalui kanal resmi.
- **Mengikuti Aturan Federasi**: Seluruh pengelola grup afiliasi wajib mematuhi dan menyelaraskan kebijakan grup dengan aturan utama yang berlaku di federasi.

**Konsekuensi**: Peringatan atau pencabutan status Admin Grup.

**Penjelasan**: Admin Grup berperan sebagai perantara utama antara komunitas lokal dan kebijakan federasi nasional. Sinkronisasi aturan sangat penting untuk menjaga keharmonisan ekosistem TCF.

*Terakhir Diperbarui: 23-12-2024*

**Aturan**:
- **Lapor Admin Federasi**: Harap segera melaporkan pelanggaran yang memerlukan tindakan FBAN kepada Admin Federasi secara tepat waktu.
- **Monitor Grup**: Pastikan grup yang dikelola tetap bersih dari konten terlarang, penipuan, dan provokasi.
- **Komunikasi Sopan**: Tetaplah bersikap sopan dan profesional saat berinteraksi dengan anggota grup.

**Konsekuensi**: Peringatan atau pencabutan status Admin Grup.

**Penjelasan**: Admin Grup adalah garda utama dalam mengelola komunitas TCF. Kegagalan dalam menjaga standar komunitas akan ditinjau kembali oleh pihak federasi.

*Terakhir Diperbarui: 28-11-2024*
