# Crypto

**Aturan**: Diskusi, promosi, atau transaksi terkait cryptocurrency tanpa izin administrasi sangat dilarang untuk menghindari potensi risiko penipuan investas.

**Konsekuensi**: Kick atau Ban/Fban.

**Penjelasan**: Kami membatasi pembahasan mengenai mata uang kripto untuk melindungi anggota dari potensi skema investasi yang merugikan (scam). Pembahasan terkait hal ini memerlukan persetujuan admin terlebih dahulu.

*Terakhir Diperbarui: 28-11-2024*
