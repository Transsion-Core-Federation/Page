# Root Module

**Aturan Penggunaan**:
- **Dilarang Penjualan Menyesatkan**: Dilarang keras menjual modul root dengan embel-embel "VVIP," "PREMIUM," "EXCLUSIVE," atau istilah sejenis lainnya.
- **Dilarang Modul Berbahaya**: Dilarang keras membuat atau mendistribusikan modul root yang dapat menyebabkan kerusakan perangkat, *brick*, *bootloop*, atau ketidakstabilan sistem. Developer wajib menguji modul sebelum dirilis.
- **Dilarang Klaim Karya Orang Lain**: Dilarang mengklaim modul root yang bukan hasil karya sendiri. Wajib memberikan kredit kepada pembuat asli.
- **Izin Jual**: Penjualan diperbolehkan hanya jika Anda adalah pembuat asli ATAU lisensi modul tersebut mengizinkan penggunaan komersial dengan mematuhi seluruh ketentuannya.
- **Kepatuhan Lisensi**: Aturan ini terhubung dengan ketentuan lisensi. Jika modul mengandung kode open source, Anda wajib mematuhi ketentuan lisensi asli (atribusi, hak cipta, dll).

**Pelanggaran & Sanksi**:
- Penjualan menyesatkan ("VVIP" dsb) = **FBAN LANGSUNG**.
- Modul berbahaya (*brick/damage*) = **FBAN INSTAN + Blacklist**.
- Pelanggaran lain = Peringatan, sanksi, atau FBAN sesuai tingkat kesalahan.
