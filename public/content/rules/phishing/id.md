# Phishing

**Aturan**: Dilarang mendistribusikan tautan atau konten penipuan (phishing) yang bertujuan untuk mencuri data pribadi anggota lain.

**Konsekuensi**: Ban atau Fban langsung.

**Penjelasan**: Keamanan akun dan data anggota sangat kami prioritaskan. Upaya pencurian kredensial (seperti password atau data sensitif) akan ditindak dengan pemblokiran permanen dari jaringan kami.
