# Promosi

**Aturan**: Anggota tidak diperkenankan mempromosikan produk, jasa, atau grup lain tanpa izin dari pihak administrasi.

**Konsekuensi**: Mute atau Kick.

**Penjelasan**: Kebijakan ini diterapkan agar grup tidak dipenuhi oleh konten iklan. Segala bentuk promosi harus melalui koordinasi dengan admin untuk menjaga ketertiban konten dalam grup.
