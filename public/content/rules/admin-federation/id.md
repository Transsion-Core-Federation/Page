# Federasi Admin

**Aturan Kerja Admin**:
- **Wajib Menyertakan Bukti**: Setiap tindakan Fban harus disertai alasan yang jelas. Admin wajib melampirkan bukti berupa tangkapan layar (screenshot), rekaman video, penerusan pesan, atau tautan bukti lainnya.
- **Tidak Membawa Masalah Personal**: Admin federasi dilarang keras menggunakan fitur Fban untuk menyelesaikan konflik pribadi atau kepentingan individu di luar aturan komunitas.
- **Bersikap Netral**: Admin tidak diperbolehkan memihak atau ikut campur dalam permasalahan anggota, kecuali jika terdapat pelanggaran aturan. Admin wajib menjadi penengah yang objektif.
- **Hak Melapor Anggota**: Anggota berhak melaporkan admin federasi yang melanggar kode etik atau aturan ini langsung kepada Founder/Owner federasi dengan menyertakan bukti yang valid.

**Konsekuensi**: Pencabutan jabatan atau Fban.

**Penjelasan**: Kami menjunjung tinggi integritas dan profesionalisme para pengelola. Admin Federasi harus menjadi teladan dalam objektivitas dan keadilan.
